# realty-post-type
Plugin that initializes a new post-type "Real Estate" and taxonomy "District".

## js ##
npm init -y

## On off running of the files
npx mix

## Watch command - await changes to the source files
npx mix watch

## production - compile all files and minify them
npx mix production

## composer install
composer install
